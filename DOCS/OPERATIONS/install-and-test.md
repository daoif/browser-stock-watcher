# 操作：安装与测试扩展

## 目的
- 把当前产物装到 Chrome / Edge 里，验证设置页、目标配置、手动检查、自动检查和通知链路。

## 前提
- 已有构建产物：`.output/chrome-mv3`
- 或者已有压缩包：`.output/browser-stock-watcher-0.1.0-chrome.zip`
- 浏览器允许加载开发者模式扩展。

## 步骤
1. 打开浏览器扩展管理页，开启开发者模式。
2. 选择“加载已解压的扩展程序”，指向 `.output/chrome-mv3`。
3. 安装后点击扩展图标，确认会直接打开一个普通标签页，地址前缀是 `chrome-extension://`，页面文件是 `options.html`。
4. 在设置页里先填一个测试目标，保存时授权对应站点的页面读取权限。
5. 点击“立即检查全部”或“只检查这个”，确认结果区能更新状态。
6. 再用一个会跳到成功 URL 的测试目标，确认命中后会收到系统通知。

## 如何确认成功
- 点击扩展图标时，直接打开扩展自己的完整标签页页面，而不是扩展按钮下面的 popup 小窗。
- 打开的地址前缀是 `chrome-extension://`，页面文件是 `options.html`。
- 保存目标后，刷新浏览器配置不丢失。
- 手动检查后，最近结果区会更新状态和最终地址。
- 命中成功 URL 时，会收到系统通知；点击通知能打开目标页面。
- 构建和打包命令可重复执行：`npm run build`、`npm run zip`
