## 新功能
- 首次发布 Browser Stock Watcher。
- 支持多目标库存监听、成功 URL 正则识别、缺货关键词识别和 Cloudflare 挑战检测。
- 支持浏览器标准通知和通知点击跳转。
- 点击扩展图标时，直接打开扩展自己的完整标签页页面。

## 交付内容
- 开发者模式加载目录：`chrome-mv3`
- Release 压缩包：`browser-stock-watcher-0.1.0-chrome.zip`

## 安装
- 下载发布产物后，按 `README.md` 和 `DOCS/OPERATIONS/install-and-test.md` 的步骤加载扩展。
