# Changelog

本文件记录每个版本的变更。

---

## v0.1.0 (2026-03-20)

### 新功能
- 完成 `Manifest V3 + Vue 3 + Tailwind` 浏览器扩展工程。
- 支持多目标监听、启停、排序、本地持久化和手动立即检查。
- 支持成功 URL 正则识别、缺货关键词识别和 Cloudflare 挑战检测。
- 支持浏览器标准通知，并在点击通知后打开目标页面。
- 点击扩展图标时，直接打开扩展自己的完整标签页页面。

### 发布
- 首次补齐 GitHub 仓库发布规范。
- 新增 `release-notes.md`、发布指南和 GitHub Actions 自动发布工作流。
